# Glass 2048

A glassmorphism-styled remake of the classic 2048 puzzle game.

**Play:** Join the numbers and get to the 2048 tile!

## Features

- Glassmorphism UI with frosted glass tiles and animated gradient background
- Responsive design (desktop & mobile)
- Score tracking with local storage
- No build tools required — pure HTML/CSS/JS
